# Nexus-Cloud-VR-for-Feebz-2.0
Nexus VR (NCVR) is an automated, cloud-based gaming ecosystem designed to bridge the gap between affordable mobile hardware and high-fidelity PC VR. By leveraging Shadow PC and Google Apps Script, NCVR transforms a standard Feebz VR Headset and smartphone into a powerful, console-like VR machine.
